Collector Hangout 4: Soup to nuts, idea to analysis

Now that you're familiar with all the core functionality of Collector we bring the pieces together. Up to this point we've been crawling but now we finally stand up and start running.

The Process in this video:

  1. Have a good experiment idea (easier said than done)
  2. Download fresh copy of Collector
  3. Load stimuli
  4. Setup Conditions file
  5. Setup procedure files
  6. Create custom trial types
  7. Setup custom scoring
  8. Dot i's and cross t's (instructions, settings)
  9. Put your experiment on the web
  10. Collect data!!!
  11. Download your data
  12. Quick tips for easy analysis of Collector data

In the course of this whirlwind session we also show how to create multisession experiments.
